## プラットフォーム概要

エミュレータは2つのハードウェアプラットフォーム構成をサポート:

| プラットフォーム | 説明 | ボードタイプ設定 |

| MVME147 | Motorola MVME147 VMEbus シングルボードコンピュータ | `"MVME147"` |

## MVME147 プラットフォーム

NetBSD/mvme68k を実行する Motorola MVME147 VMEbus シングルボードコンピュータをエミュレート。

### メモリマップ

| アドレス範囲 | サイズ | 説明 |
|-------------|--------|------|
| 0x00000000 - (設定可能) | 最大 32 MB | メインRAM |
| 0xFF800000 - 0xFFBFFFFF | 4 MB | ROM（オプション） |
| 0xFFFE0000 - 0xFFFE07FF | 2 KB | MK48T02 NVRAM/RTC |
| 0xFFFE1000 - 0xFFFE102F | 48 B | PCC（周辺チャネルコントローラ） |
| 0xFFFE1800 - 0xFFFE1803 | 4 B | LANCE イーサネットコントローラ |
| 0xFFFE3000 - 0xFFFE3003 | 4 B | Z8530 SCC（シリアルコントローラ） |
| 0xFFFE4000 - 0xFFFE4001 | 2 B | WD33C93 SCSI コントローラ |
| 0xFFFE0000 - 0xFFFEFFFF | 64 KB | I/O 空間キャッチオール（フォールバック） |

### 割り込みルーティング

全デバイス割り込みは PCC を経由:

```
Z8530 SCC  ──→ PCC ICR ($26) ──→ CPU IPL
LANCE      ──→ PCC ICR ($28) ──→ CPU IPL
WD33C93    ──→ PCC ICR ($2A) ──→ CPU IPL
Timer 1    ──→ PCC ICR ($18) ──→ CPU IPL
Timer 2    ──→ PCC ICR ($1A) ──→ CPU IPL
```

### PCC（周辺チャネルコントローラ）

| 項目 | 詳細 |
|------|------|
| ファイル | `IO/PccDevice.cs` |
| チップ | MVME147 PCC（カスタム ASIC） |
| ベースアドレス | 0xFFFE1000 |
| サイズ | 48 バイト (0x00-0x2F) |
| 状態 | 完全実装済み |

機能:
- **2 x 16ビットタイマー**: 160 kHz（壁時計ベース、CPU 速度非依存）
- **DMA コントローラ**レジスタ（テーブルアドレス、データアドレス、バイトカウント、データホールド）
- **12 割り込み制御レジスタ** (ICR): 全 MVME147 デバイス用
- **ハードウェアリセット**: RESET 命令 (0x4E70) 経由

レジスタマップ:

| オフセット | レジスタ | 説明 |
|-----------|---------|------|
| $00-$03 | DMA Table Addr | DMA テーブルポインタ |
| $04-$07 | DMA Data Addr | DMA データポインタ |
| $08-$0B | DMA Byte Count | DMA 転送サイズ (24ビット) |
| $0C-$0F | DMA Data Hold | DMA データホールドレジスタ |
| $10-$11 | Timer 1 Preload | タイマー1 初期値 |
| $12-$13 | Timer 1 Count | タイマー1 現在値 |
| $14-$15 | Timer 2 Preload | タイマー2 初期値 |
| $16-$17 | Timer 2 Count | タイマー2 現在値 |
| $18 | Timer 1 ICR | タイマー1 割り込み制御 |
| $19 | Timer 1 Control | タイマー1 有効/COC/オーバーフロー |
| $1A | Timer 2 ICR | タイマー2 割り込み制御 |
| $1B | Timer 2 Control | タイマー2 有効/COC/オーバーフロー |
| $26 | SCC ICR | Z8530 割り込み制御 |
| $28 | LANCE ICR | AM7990 割り込み制御 |
| $2A | SCSI ICR | WD33C93 割り込み制御 |
| $2C | SW1 ICR | ソフトウェア割り込み 1 |
| $2D | Vector Base | 割り込みベクタベース（デフォルト 0x40） |
| $2E | SW2 ICR | ソフトウェア割り込み 2 |

ICR フォーマット: INT (bit 7) | IEN (bit 3) | IL[2:0] (bits 2-0)

### Z8530 SCC（シリアル通信コントローラ）

| 項目 | 詳細 |
|------|------|
| ファイル | `IO/Z8530Device.cs` |
| チップ | Zilog Z8530（デュアルチャネル SCC） |
| ベースアドレス | 0xFFFE3000 |
| サイズ | 4 バイト |
| 状態 | 完全実装済み |

機能:
- **デュアルチャネル**: チャネル A（コンソール）、チャネル B（補助）
- **16レジスタ制御モデル**（チャネルごとに WR0-WR15, RR0-RR15）
- **RX FIFO** 割り込みサポート付き
- **TX シミュレーション**: アイドル追跡と定期的な割り込み再アサート
- **DCD/CTS** 常時アサート（端末接続状態）

アドレスマップ:

| オフセット | レジスタ |
|-----------|---------|
| $0 | チャネル B 制御 |
| $1 | チャネル B データ |
| $2 | チャネル A 制御 |
| $3 | チャネル A データ |

### MK48T02 NVRAM/RTC

| 項目 | 詳細 |
|------|------|
| ファイル | `IO/Mk48t02Device.cs` |
| チップ | Mostek MK48T02 (2 KB SRAM + RTC) |
| ベースアドレス | 0xFFFE0000 |
| サイズ | 2048 バイト |
| 状態 | 完全実装済み |

構造:
- **0x0000-0x07F7**: 汎用 SRAM (NVRAM ストレージ)
- **0x07F8-0x07FF**: クロックレジスタ (BCD フォーマット)

クロックレジスタ:

| オフセット | レジスタ | フォーマット |
|-----------|---------|-------------|
| $7F8 | Control | bit 7: Write, bit 6: Read |
| $7F9 | Seconds | BCD 0-59 |
| $7FA | Minutes | BCD 0-59 |
| $7FB | Hours | BCD 0-23 |
| $7FC | Day of Week | 1-7 |
| $7FD | Date | BCD 1-31 |
| $7FE | Month | BCD 1-12 |
| $7FF | Year | BCD 0-99（基準年 1968） |

MVME147 用プリセット値:
- $0774: オンボード RAM 終端+1 (32ビット BE)
- $0778-$077A: イーサネット MAC アドレス (08:00:3E プレフィックス)

### WD33C93 SCSI コントローラ

| 項目 | 詳細 |
|------|------|
| ファイル | `IO/Wd33c93Device.cs` |
| チップ | Western Digital WD33C93 (SCSI Level I) |
| ベースアドレス | 0xFFFE4000 |
| サイズ | 2 ポート |
| 状態 | 完全実装済み |

機能:
- **フェーズ単位プロトコル**: MSG_OUT -> CMD -> DATA_IN/OUT -> STATUS -> MSG_IN
- **32 内部レジスタ** ($00-$1F、間接アクセス)
- **PIO / DMA 転送**モード
- **SBT（シングルバイト転送）**モード
- **最大 8 SCSI ターゲット** (ID 0-7)

ポートアクセス:

| オフセット | 読出し | 書込み |
|-----------|-------|-------|
| $0 | ASR（補助ステータス） | アドレスレジスタ |
| $1 | データポート | データポート |

ASR ビット: INT (7), BSY (5), CIP (4), PE (1), DBR (0)

コマンド:

| コード | コマンド | 説明 |
|--------|---------|------|
| 0x00 | RESET | コントローラリセット |
| 0x01 | ABORT | 現在のコマンド中断 |
| 0x04 | DISCONNECT | 接続切断 |
| 0x06 | SEL_ATN | アテンション付きターゲット選択 |
| 0x08 | SEL_ATN_XFER | 選択 + 自動転送 |
| 0x20 | XFER_INFO | フェーズベースデータ転送 |
